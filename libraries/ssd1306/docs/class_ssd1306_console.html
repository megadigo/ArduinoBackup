Detailed Description

Ssd1306Console represents object to work with LCD display. Easy to use:

Ssd1306Console console;
void setup()
{
ssd1306_128x64_spi_init(3, 4, 5);
ssd1306_clearScreen();
console.print( "Hello" );
}
